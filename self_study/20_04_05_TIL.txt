
오늘도 강조...! html 로는 항상 의미와 구조에 대해서 집중하자.
인라인 텍스트
    <a>
        anchor
        외부 문서로 연결할 수 있는 태그
        다른 URL로 연결할 수 있는 하이퍼링크를 설정
        속성
            href: 링크 URL (생략가능 하지만 필수처럼 활용해야한다.)
            target: 링크 URL의 표시(브라우저 탭) 위치
                _self(현재 창에 띄우겠다), _blank(새로운 탭을 띄우겠다.)
            rel: 현재 문서와 링크 URL의 관계 (값 : license, prev, next ...)
            /// MIME 타입은 마임이라고 읽는군...
    
        inline 요소는 가로 세로 값을 조정할수 없다. layout보다는 텍스트를 다루는 요소.
            <a> 는 inline이라 힘들기 때문에 , a{ display: block;} 으로 시도 해보자.
        
        <a> 는 같은 페이지 위치로도 이동시킬 수 있다. 이것은 어떤 의미인가?
            !!! 요소에 id를 부여하여 해당 위치로 이동시킬수 있다 !!!
            !!! id 선택자는 # (hash) !!!

    <abbr>
        abbreviation. 약어.
            <abbr title="HyperText Markup Language">HTML</abbr>
    
    <b> <mark> <strong>
        (필요 한지 모르겠다.)

    <em>
        ' 의미 강조 ' 를 표시
        중첩 가능, 중첩될수록 강조 의미가 강해짐.
        기본적으로 이탤릭체로 표시됨.
        !!! 정보통신보조기기 등에서 구두 강조로 발음됨. (브라우저 등)

    <i>
        icon이나 특수문자를 가져올 때 사용하면 좋다.
    
    <span>
        본질적으로 아무것도 나타내지 않는 콘텐츠 영역을 설정
        <div>와 유사하게 사용 가능 (예를 들어, css를 입힐때. 꾸밀때.)
            다만, div는 블록 요소. span은 인라인 요소.
            span { display: inline; }

        - 인라인 요소는 텍스트를 다루는데 적합.
        - 블록요소는 레이아웃을 다루는데 적합

        - 클래스 선택자 기억나니? '.' 이야

    <br/>
        빈 태그. 줄 바꿈을 설정.
    
    <del>
        삭제된 텍스트의 범위를 지정

    <ins>
        추가된 텍스트의 범위를 지정

    이외 인라인 텍스트 부분은 페이지만 참고하여 공부하면 되겠다.
